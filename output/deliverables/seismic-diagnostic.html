1. Calculation Procedure Overview

The calculator determines seismic design forces on nonstructural components per ASCE 7-22 Chapter 13, then checks anchor capacity per ACI 318-19 Chapter 17. Three seismic approaches are supported, defaulting to the conservative general case.

2. Seismic Force Equations — ASCE 7-22 §13.3

2.1 Seismic Design Force

ASCE 7-22 Eq. 13.3-1
Seismic Design Force
$$F_p = 0.4 \times S_{DS} \times I_p \times W_p \times \frac{H_f}{R_\mu} \times \frac{C_{AR}}{R_{po}}$$
$S_{DS}$
Design spectral acceleration at short period (g) — from USGS API or manual entry
$I_p$
Component importance factor — Table 13.3-1 (1.0 for RC I/II/III, 1.5 for RC IV)
$W_p$
Component operating weight (lbs)
$H_f$
Height amplification factor — §13.3.1.1
$R_\mu$
Structure ductility reduction factor — §13.3.1.2
$C_{AR}$
Component resonance ductility factor — Tables 13.5-1 / 13.6-1
$R_{po}$
Component strength factor — Tables 13.5-1 / 13.6-1
Eq. 13.3-2
Minimum
$$F_{p,\min} = 0.3 \times S_{DS} \times I_p \times W_p$$
Eq. 13.3-3
Maximum
$$F_{p,\max} = 1.6 \times S_{DS} \times I_p \times W_p$$

Governing: $F_{p,\text{design}} = \min\!\Big(\max(F_p,\; F_{p,\min}),\; F_{p,\max}\Big)$

2.2 Height Amplification Factor (Hf)

Program Input: SiteTab → Building Height (h), Attachment Height (z), Period (Ta — optional, under Known SFRS)

At or below grade (z ≤ 0):

$H_f = 1.0$

Eq. 13.3-4
With Known Period
$$H_f = 1 + a_1 \left(\frac{z}{h}\right) + a_2 \left(\frac{z}{h}\right)^{10}$$
$a_1$
$= \min\!\left(\dfrac{1}{T_a},\; 2.5\right)$
$a_2$
$= \max\!\left(0,\; 1 - \left(\dfrac{0.4}{T_a}\right)^2\right)$

Used when Known SFRS approach is selected and Ta > 0.

Eq. 13.3-5
Without Period (Default / Conservative)
$$H_f = 1 + 2.5 \left(\frac{z}{h}\right)$$

Used in General Case approach (always), or in Known SFRS when Ta is not provided. Note: z/h is capped at 1.0.

ASCE 7-16 vs 7-22: The old formula used a factor of 2: $H_f = 1 + 2(z/h)$. ASCE 7-22 increased this to 2.5, making the default 25% more conservative at the roof level.

2.3 Structure Ductility Reduction Factor (Rμ)

Program Input: SiteTab → Seismic Approach radio → Known SFRS dropdown (R, Ω0, Ie)

At or below grade (z ≤ 0):

$R_\mu = 1.0$

Eq. 13.3-6
Above Grade — Known SFRS
$$R_\mu = \max\!\left(1.3,\;\; \sqrt{\frac{1.1 \times R}{I_e \times \Omega_0}}\right)$$
$R$
Response modification coefficient of building SFRS (Table 12.2-1)
$\Omega_0$
Overstrength factor of building SFRS (Table 12.2-1)
$I_e$
Building importance factor (Table 1.5-2) — auto-populated from Risk Category

General Case default: $R_\mu = 1.3$ (above grade). This is the minimum per Eq. 13.3-6 and is used when the building SFRS is unknown.

New in ASCE 7-22: Rμ replaces the implicit ductility in the old Rp factor. Buildings with special moment frames (R = 8) can achieve Rμ ≈ 1.71, significantly reducing component forces compared to the 1.3 default.

2.4 Floor Acceleration Alternative

§13.3.1.1 Alt.
Floor Acceleration Method
$$F_p = A_i \times \frac{C_{AR}}{R_{po}} \times I_p \times W_p$$

Where $A_i$ is the floor acceleration at the equipment level (g) from structural analysis. Same Fp,min and Fp,max bounds apply.

Program Input: SiteTab → Floor Acceleration radio → Ai field (g). This replaces the Hf/Rμ equation-based approach entirely.

4. SDC Exemption Logic — §13.1.4

Program Display: SiteTab → SDC badge (green/amber/red) after USGS lookup. Informational only — does not block calculation.

The calculator checks §13.1.4 Items 1–6 in order and returns the first matching exemption:

Item Condition Result
1 SDC = A EXEMPT All nonstructural components
3 SDC B/C + M/E/P + Ip = 1.0 + Wp ≤ 20 lbs EXEMPT Lightweight component
2 SDC B/C + M/E/P + Ip = 1.0 + z/h ≤ 2/3 EXEMPT Below upper third
5 SDC B + Architectural + Ip = 1.0 EXEMPT Fastened to structure
6 SDC C + Architectural + Ip = 1.0 + z ≤ 0 EXEMPT At/below grade
— None of the above (including all SDC D/E/F) REQUIRED Full analysis

Note: Item 3 is checked before Item 2 because a 15-lb component at z/h = 0.9 should still be exempt under the weight criterion. Item 4 (furniture/similar) is not implemented as it requires subjective classification.

5. Overturning Moment Analysis

CRITICAL: Direction 1 (force along LENGTH) uses moment arm = L/2. Direction 2 (force along WIDTH) uses arm = W/2. Do NOT swap these arms.
Overturning Moment (both directions)
$$M_{OT} = F_p \times h_{cg}$$

Where $h_{cg}$ = center of gravity height above base, converted to feet.

Resisting Moments — 0.9W Dead Load Factor
$$M_{R1} = 0.9 \times W_p \times \frac{L/2}{12} \quad \text{(Direction 1: force along length)}$$ $$M_{R2} = 0.9 \times W_p \times \frac{W/2}{12} \quad \text{(Direction 2: force along width)}$$
Net Uplift Moments
$$M_{\text{net},1} = \max(0,\; M_{OT} - M_{R1})$$ $$M_{\text{net},2} = \max(0,\; M_{OT} - M_{R2})$$

Governing direction = whichever produces higher tension per anchor (considering anchor spacing).

5.1 Anchor Demands with Ω0p

Ω0p is per-component from Tables 13.5-1/13.6-1 (not a blanket 2.0). This overstrength factor is applied to anchor demands to ensure ductile failure mode.
Tension Demand Per Anchor
$$T_u = \frac{M_\text{net} / s_\text{spacing}}{n_\text{tension}} \times \Omega_{0p}$$
$s_\text{spacing}$
Anchor spacing in governing direction (ft)
$n_\text{tension}$
Number of anchors in tension row
$\Omega_{0p}$
Component anchorage overstrength factor
Shear Demand Per Anchor
$$V_u = \frac{F_p \times \Omega_{0p}}{n_\text{total}}$$

5.2 Vertical Seismic Effect

§12.4.2.2
Vertical Seismic Force Component
$$E_v = \pm\, 0.2 \times S_{DS} \times D$$

For uplift check, net stabilizing gravity = $(0.9 - 0.2 \times S_{DS}) \times W_p$. Used in Rigid Bolt Group analysis.
